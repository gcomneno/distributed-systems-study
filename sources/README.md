# Sources Policy

## Primary source

Sukumar Ghosh, _Distributed Systems: An Algorithmic Approach_,
Second Edition, CRC Press, 2015.

## Repository policy

This directory records bibliographic references and source notes only.

It must not contain:

- copyrighted books or PDFs;
- copied chapters;
- extensive quotations;
- complete solutions reproduced from external sources;
- private study material.

External material must be clearly distinguished from interpretation,
practical examples, and original exercises.
